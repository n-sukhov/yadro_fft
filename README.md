### Задача 2
Написать на С++ класс быстрого прямого и обратного преобразования Фурье комплексных значений с возможной длиной преобразования кратной 2, 3, 5.
Запустить для случайных комплексных входных данных сначала прямое, а потом обратное преобразование Фурье.
Сравнить ошибку между входными и выходными данными.

---

### Сборка проекта

Для сборки проекта необходимо перейти в папку проекта и выполнить:

```sh
mkdir build
cd build
cmake ..
cmake --build .
```

---

##### Автоматический режим (запуск с N, записанным в коде программы)

*(N - количество комплексных значений, по заданию должно раскладываться только на простые множители 2, 3, 5)*

Для запуска задания необходимо выполнить:

Linux / macOS:

```sh
./yadro_fft
```

Windows:


```bat
yadro_fft.exe
```

##### Ввод параметра N вручную

Linux / macOS:

```sh
./yadro_fft <N>
```

Windows:

```bat
yadro_fft.exe <N>
```

В результате выполения программы выводится средняя абсолютная ошибка, максимальная ошибка, происходит сохранение сгенерированного сигнала длиной $N$, спектра, полученного прямым БПФ, и результат обратного БПФ в csv-файлы в папку `data/` для дальнейшей визуализации на Python.

*Пример запуска программы:*
```shell
ns@Vivobook:~/Desktop/yadro_fft$ ./yadro_fft 600                                                    
FFT length N = 600                                                                                  
MAE = 9.56278375e-16                                                                                
Max absolute error = 4.09430021e-15                                                                 
ns@Vivobook:~/Desktop/yadro_fft$ ./yadro_fft                                                        
FFT length N = 36000                                                                                
MAE = 1.24915112e-15                                                                                
Max absolute error = 6.24716291e-15    
```
